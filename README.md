## About the Project

Inspired by the YouTube channel 'Geekble', I developed a custom security keyboard tailored specifically for the Hanam Youth Center.

## Components & Tools

**Materials (BOM):**
* 1x Arduino pro micro (C Type)
* 20x Kailh Mechanical Switches
* 20x Hot-swap Sockets
* 0.1mm ~ 1mm Enamel Wire
* 25x M2*L10

**Equipment:**
* 3D Printer (Multi-color capability required)
* Soldering Iron
* Tweezers

## How to Use

* **Software & Wiring (`.ino`):** All details regarding the Arduino wiring diagram, code upload instructions, and required library versions are documented as comments inside the `HK-1 Code.ino` file.
  
* **Hardware Assembly (`.f3d`):** For detailed assembly instructions and 3D modeling structures, please open and inspect the `HK-1 3D Models.f3d` (Fusion 360) file.
